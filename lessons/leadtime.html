Lead time: quanto prima prenotano

Obiettivo: stimare per mese il lead time medio e i giorni necessari a coprire il 50% e il 75% delle prenotazioni.

Definizioni

lead.time = giorni prima dell'inizio noleggio in cui il cliente ha effettuato la prenotazione.

Statistiche mensili

library(dplyr)
library(lubridate)

bringo_stat <- bringo |>
  mutate(mese = month(Data.Inizio, label = TRUE, abbr = FALSE)) |>
  group_by(mese) |>
  summarise(
    lead_time_medio = mean(lead.time, na.rm = TRUE),
    n_prenotazioni = n()
  )

50% e 75% prenotazioni

bringo_stat <- bringo |>
  mutate(mese = month(Data.Inizio, label = TRUE, abbr = FALSE)) |>
  group_by(mese) |>
  summarise(
    lead_time_medio = mean(lead.time, na.rm = TRUE),
    giorni_50_percento = quantile(lead.time, 0.50, na.rm = TRUE),
    giorni_75_percento = quantile(lead.time, 0.75, na.rm = TRUE),
    n_prenotazioni = n()
  )
